# ChooseAdventure BE

### Requirements
- Postgres (https://hub.docker.com/_/postgres) 
```sh
    $ docker run --name yourName -e POSTGRES_PASSWORD=yourPassword -p 5432:5432 -d postgres:yourVersion
```
- Redis (https://hub.docker.com/_/redis)
```sh
    $ docker run --name yourName -d redis
```
- That's it!
